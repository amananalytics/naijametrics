Your FYP data, cleaned into a readable public-facing format with interactive quarterly views and separation patterns.

03 Methodology & Sources
📊
Descriptive Statistics
Job Creation Rate (JCR), Job Separation Rate (JSR) and Net Employment Change computed per occupational category. Quarterly breakdown across Q1–Q3 2024 from the NBS Business Establishment Survey.
χ²
Chi-Square Homogeneity
Tests whether the distribution of job creation and separation across occupational categories is statistically homogeneous. Full analysis performed in base R with contingency table output.
🌐
World Bank Open Data API
Macro indicators fetched live from api.worldbank.org/v2/country/NG at page load. Data is rendered dynamically and the source code is fully open on GitHub Pages.
